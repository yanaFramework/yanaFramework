Yana Manual

deutsche Ausgabe

Diese Software richtet sich an KMUs, freiberufliche Softwareentwickler und Privatleute. Es ist eine Plattform für die Entwicklung von Programmen für das Internet, ein Firmennetzwerk oder Desktoprechner. Aktuelle Verfahren der Softwaretechnik, welche die Arbeit vereinfachen oder beschleunigen, werden unterstützt. Zum Beispiel: Rapid-Prototyping oder der Aufbau von Softwareproduktlinien.
Für Privatleute gibt es eine Reihe vorinstallierter Anwendungen, welche Sie ohne Programmierkenntnisse sofort nutzen können.
In diesem Handbuch wird die Installation, Konfiguration und Bedienung der Software beschrieben.

Stichwortsuche

Dieses Handbuch verfügt über eine Suchfunktion, welche es Ihnen ermöglicht, alle Texte nach Stichworten zu durchsuchen.
Bitte beachten Sie: Um diese Funktion nutzen zu können, müssen Sie in Ihrem Browser Java und JavaScript aktiviert haben.
