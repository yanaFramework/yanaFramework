Search Engine

How does it work?

To search for one or more words, insert these words separated by spaces. It is not case sensitive, so you don't have to care about that.
Click on "start". After that, you get a list of pages, that contain hits for all words you have entered.

The list will show the name and a brief description for each page. When you click on of these pages, you will be relocated to the corresponding web- document.
